# Windows Server 2025 Active Directory Lab

## Overview
This project demonstrates the setup and management of an Active Directory environment using Windows Server 2025 in a VirtualBox virtual machine.

## Technologies Used
- Windows Server 2025
- Active Directory Domain Services (AD DS)
- VirtualBox

## Key Features / Tasks Completed
- Installed and configured Active Directory Domain Services
- Created and managed users and groups
- Configured Organizational Units (OUs)
- Implemented Group Policies (password policies)
- Assigned roles and delegated permissions
- Performed user account management (add/remove/reset passwords)

## Setup Process
  1. Installed Windows Server 2025 on VirtualBox
  2. Promoted server to Domain Controller
  3. Configured AD DS roles
  4. Created domain envrionment and users
  5. Applied group policies and security configurations
 
## Learning Outcomes
  1. Gained hands-on experience with enterprise-level identity management
  2. Learned how to configure and manage Active Directory envrionments
  3. Developed skills in user administration and enforcing security policies
